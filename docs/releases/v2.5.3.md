# Hermes Desktop v2.5.3

**Release date:** 2026-04-28  
**Portable asset:** `HermesDesktop-portable-x64.zip` (from [GitHub Releases](https://github.com/RedWoodOG/Hermes-Desktop/releases))

## Highlights

### Brand refresh

- Replaced the app package icon, store logo, square tile assets, wide tile asset, lock screen logo, and splash screen with the new gold Hermes mark.
- Updated the README logo to match the shipped app identity.
- Added a visible app version row to the Dashboard System card.

### Engineering

- Versioning: `HermesDesktop` assembly and `Package.appxmanifest` identity set to **2.5.3.0**.

## Upgrade notes

- Existing users do not need to change configuration.
- After downloading the portable zip, open Hermes and check **Dashboard -> System -> Version** to confirm the running build.
