# Hermes Desktop v2.5.7

Release date: 2026-05-09

## Highlights

- **Buddy is no longer ASCII-only.** The Buddy page and side panel now render a native WinUI vector companion.
- **Character crafting.** Users can choose species, palette, eyes, and accessory before hatching, then adjust the visual look later without rerolling identity, rarity, or stats.
- **Persistent visual traits.** Crafted choices are saved to `buddy.json` alongside the Buddy soul data.
- **Offline-safe art direction.** The avatar is rendered locally in WinUI, avoiding remote image/API dependency while following the composable-avatar pattern found in open-source avatar systems.

## Verification

- `dotnet build Desktop\HermesDesktop\HermesDesktop.csproj --no-restore -c Debug -p:Platform=x64`
- `dotnet test Desktop\HermesDesktop.Tests\HermesDesktop.Tests.csproj --no-restore`
